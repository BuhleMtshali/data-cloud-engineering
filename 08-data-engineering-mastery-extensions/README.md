## 🧠 Key Skills I’m Building

### 🧪 Data Quality Drills
Practicing checks for missing values, duplicates, schema issues, freshness, and invalid records.

### 🧱 Data Modeling Drills
Improving fact tables, dimension tables, star schemas, and data dictionaries.

### ⚙️ Pipeline Operations
Practicing reruns, backfills, failures, logging, monitoring, and recovery notes.

### 🧠 Advanced SQL
Building stronger query confidence through joins, CTEs, window functions, and validation queries.

### 🐍 Python Pipeline Hardening
Improving scripts with logging, config, tests, error handling, and cleaner structure.

### 💼 Portfolio Polish
Improving README files, diagrams, project explanations, and interview stories.
